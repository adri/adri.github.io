MacOS workflows and tools

By Adrian Philipp

Tools I use on macOS.

  1. macOS
    • US keyboard layout with German Umlauts via Option Key
    • Homebrew with Brewfile to manage installed software via a configuration
    • Sensible defaults as needed
  2. Terminal
    To run commands I use iTerm.
    • Fish shell
      This shell has great autosuggestions and good performance. I switched to it because all my zsh plugins slowed down the terminal. Fish has great defaults.
    • itermocil
      To open up multiple split windows in a pre-defined configuration.
    • Show command times 2013-08-07-shell-show-time-last-command-took
    • Git-number to quickly refer to files by numbers
  3. Knowledge management
    • Obsidian is great for writing notes and managing todos
    • To find information I’m Building a Memex
    • Whimsical for beautiful flow charts
  4. Little helpers
    • Rocket app to quickly type emoij
    • Dozer to clean up the menu bar
    • Turbo Boost Switcher Pro to control the fan noise, see How to avoid loud fans on MacBook Pro
    • A
  5. Security and privacy
    • Little Snitch Firewall to keep tabs on apps internet connections
    • Gas Mask To block requests to tracking websites etc by adding them to the hosts file. Example block lists
    • 1Password to manage passwords until iCloud passwords can be shared, see Personal email and authentication setup
    • Wipr good and fast ad-block
  6. Others
    • ReadKit to have all reading in one place, see Reading news
    • MoneyMoney to manage personal finance in one place
    • Ferdi to group work Slack and email and manage interruptions
